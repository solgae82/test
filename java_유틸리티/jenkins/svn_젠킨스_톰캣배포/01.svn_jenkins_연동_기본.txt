------------------- 연동설정 순서

1. jenkins 수동 설정으로 svn 접근해서 메이븐 플젝 빌드하기
2. svn 훅 트리거로 jenkins 호출해서 메이븐 플젝 빌드하기 (토큰필요)
3. jenkins 에서 빌드한 war 파일 톰캣 서버에 배포하기

svn 커밋(변경)이 되면 > svn이 jenkins를 변경 호출을 알리고 > jenkins은 svn 플젝 소스를 가져와 빌드 > 마지막 톰캣서버에 배포

------------------- 기본 설정 및 1. jenkins 수동 설정으로 svn 접근해서 메이븐 플젝 빌드하기

젠킨슨 버전 : Jenkins 2.528.3

ID(이름설정) : 관리자 

Plugins
	Maven Integration plugin
	버전3.27

	Subversion Plug-in
	버전1303.vcfd9679fb_c12
	
	Deploy to containerVersion
	1.17

jenkins 관리 > Tools
	Maven Configuration
		Default settings provider(선택) : Setting file in filesystem
		File path(입력) : D:\sts\maven\apache-maven-3.6.3\conf\settings.xml
		
	JDK installations
		Name(입력) : openJDK17
		JAVA_HOME(입력) : D:\sts\jdk\jdk-17.0.3.1
		
	Git installations
		Name(입력) : Default
		Path to Git executable : C:\Program Files\Git\bin\git.exe
		
	Maven installations
		Name(입력) : maven3.6.3
		MAVEN_HOME : D:\sts\maven\apache-maven-3.6.3
		
(사용자:관리자) > Security > API Token
	Add new token > 새 토큰 생성
		: 토큰이름 영어로 짓고, 생성한 토큰은 복사해서 보관한다.
		

Item (새 빌드) 생성하기 
새로운 Item > New Item > enter an item name : 빌드이름 입력 >  Maven project 선택 > OK버튼
	General
	
	소스 코드 관리 > Subversion 선택 > Modules
		Repository URL(입력) : svn://192.168.0.6/repo1/web
		Credentials > + Add 버튼으로 생성 > (Jenkins Credentials Provider: Jenkins) 
		Add Credentials
			Domain(선택) : Global credentials 9 unrestricted)
			Kind(선택) : Username with password
				Scope(선택) : Global (jenkins, nodes, items, all child items, etc)
				Username(입력) : 서브버전 ID
				Password : 서브버전 password
				ID(입력) : Jenkins 내부에서 쓰는 자격증명 고유 ID명 입력(비워두면 자동생성됨.명시적으로 입력하는게 유리)
				Description(입력) : 이 자격증명에 대한 설명 기입
		Local module directory(입력) :  비운다
		Repository depth(선택) : infinity(기본)
		Ignore externals : 뭔지 모르지만 디폴트(체크)된 상태로 놔둠
		Cancel process on externals fail : 뭔지 모르지만 디폴트(체크)된 상태로 놔둠
		
		나머지 다 디폴트
		
	Triggers
		빌드를 원격으로 유발 (예:스크립트 사용) : 체크
		Authentication Token : 관리자에서 만든 API Token 값을 넣는다
		(아래에 빌드 유발 방법이 url과 함께 자세히 나온다, 잘 읽어보면 좋다)
		
	Build
		Root POM(입력) : pom.xml
		(잘못된 위치라면 빨간색 에러 표시가 나타난다)
		
		
	Goals and options(입력) : clean package
		(메이븐 Goal을 쓰면 된다, clean package 또는 clean install)
		
			
	Apply & Save 버튼 적용.
	
새 Item을 생성했으니 '지금 빌드(build Now)'  선택해서 수동 빌드를 테스트해보자.
