기본적으로 jenkins는 java was에 돌아가는 웹애플리케이션이다
centOS7 에 설치한다


1. https://www.jenkins.io/download/
	여기 가면 운영체제별로 설치법 잘 나온다
	설명대로 리눅스 yum 으로 해도 된다.
	
	그러나 나는 톰캣이 이미 설치되어 있으므로
	jenkins.war 로 받는다

2. 톰캣 home 에 업로드

3. 톰캣기동
	기동시 톰캣 로그를 보면, 설치 패스워드가 나온다

4. /jenkins 접속

	설치페이지가 뜨는데, 패스워드 입력창이 나올 것이다
	페이지 설명대로 
		/root/.jenkins/secrets/initialAdminPassword 
	에 가서 복사 하거나 또는
	3번 톰캣 기동시 로그에서 보았던 패스워드를 복사(같은 패스워드)
	해서 입력 창에 넣고 다음 진행

5. 설치 모드
	설치모드 가 나오는데, 기본 설치 또는 커스텀이다..
	기본설치로 하니 플러그인들을 엄청 다운로드 받는다(시간걸림)

	그러므로 커스텀으로 하는게 나을 것같다

6. 설치 마무리
	계정 , 패스워드, 이름, 이메일 주소 입력등을 받는다
	(꼭 제로보드 설정하는 거 같다)

7. jenkins 관리 홈이 접속 된다.

-------------------

jenkins.war 는 서버없이 java로 독립 실행 시킬 수 있다

	java -jar jenkins.war --httpPort=9090
	
	http://localhost:9090/ 접속
	
실행파일 만들기
	jenkins.bat 
		@echo off
		REM 젠킨스 설치 경로로 이동 (예시: C:\Jenkins)
		cd "D:\programs\jenkins"

		REM 젠킨스 실행 (백그라운드에서 실행, 창이 뜨지 않음)
		start "" java -jar jenkins.war --httpPort=9090
