동작모드
1. 무선익스텐더(AP+Client) : 무선Client 기능(확장기기능) + 무선AP 기능 활성화
2. 무선Client : 확장기(extender) 기능.
3. 무선AP : 인터넷 무선 공유기 기능. 
	
	1. 인터넷 선 - 확장기(extender) 랜포트 연결 = 무선공유기(AP)
	2. 인터넷 공유기 - 랜선 - 확장기(extender) 랜포트 연결 = 무선공유기(AP)
	3. 인터넷 무선공유기 - 확장기(extender) 연결 =  무선공유기(AP)

	SSID 및 패스워드를 설정해서 사용한다.
	인증 암호화는(WPA2PSK + AES)를 설정한다.

		
결론 : 무선공유기 형식으로 쓰고 싶으면 3번 무선AP 설정을 하면 된다. (또는 1번처럼 두 기능 모두 활성화 하던지)


여러가지 경우에 수 테스틀 해 본 결과,
이 기기는 기본이 무선공유기고, 부가 기능으로 wifi 확장기능이 있다고 봐도 될 것같다.

확장기 설치하는 법
	1. WPS 기능으로 자동연결
		- 일단 무조건 초기화
		- 연결할 무선공유기 WPS 누르고, 확장기 WPS 눌러서 서로 연결해서 설치 후 해당 음영지역(wifi 신호가 약한 곳) 근처에서 다시 확장하면 됨.
	
	2. 무선 모바일 기기에서 설정하기
		- 일단 무조건 초기화
		- iptime_extender_setup 이라는 ssid가 나올 것임.(초기화 해야 나옴), 연결
		- 웹브라우저로 설정화면 들어가야 되는데 설정주소는 http://setup.iptime.com (또는 192.168.0.254)
		- 설정화면에서 
			확장할 무선네트워크 : 무선공유기(연결할) SSID 선택 
			네트워크 암호 : 무선공유기(연결할) SSID 패스워드
			
			---------------------------------
			만일! 
			확장기가 아닌 무선AP모드로 쓰고 싶을땐, 인터넷 선을 랜입력 포트에 연결한 후  
				확장할 무선네트워크 : 이 확장기를 무선공유 제공할 SSID명 입력.
				네트워크 암호 : 무선공유 제공할 네트워크 암호입력.
			---------------------------------

		- 관리자계정 : 
			- 이 설정화면을 다시 들어올때 로그인/패스워드 후 들어 올 수 있게 해놓은 설정.(무조건 해야함)
			- 이 설정이 있다는 것은 다시 다른 확장 SSID를 설정할 수 있다는 얘기..
			하지만 설명하기 귀찮으니까 무조건 초기화 후 다시 설정하는 것으로 일단 이해.


	3. pc로 연결 설정하기
		- 일단 무조건 초기화
		- 랜선으로 확장기와 pc 연결
		- 웹브라우저로 설정화면 들어가야 되는데 설정주소는 http://setup.iptime.com (또는 192.168.0.254)
		- 익스텐더 설정 메뉴 선택
		- AP검색에서 확장할 무선SSID 선택
		- 동작모드는 문서 제일 위 동작모드 설명 참조
			- 확장기만 사용하려면  '무선Client' 선택, 
			무선공유기처럼 쓰려면 '무선AP' 선택, 
			두 기능 다 활성화 하려면 무선익스텐더(AP+Client)
