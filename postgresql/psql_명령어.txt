주요 psql 접속 명령어

	-h : 접속 IP 주소
	-p : 접속 port 번호
	-U : 접속 계정
	-d : 접속 DB명
	-c : psql 실행 시 SQL 명령어를 1회 실행하고 종료하는 CLI 옵션입니다 
		(psql -c "SELECT ...")
	------
	
	기본 접속 (로컬): psql -U postgres
		
		사용자->postgres , DB->postgres
		
	특정 DB 접속: psql -U [사용자] -d [DB명]
	
		psql -U dong2025 -d postgres
		
	원격 서버 접속: psql -h [호스트IP] -p [포트] -U [사용자] -d [DB명]

		psql -h 192.168.111.7 -p 5432 -U dong2025 -d postgres
		
	
	Linux/Unix/Windows 패스워드 입력: 명령어 실행 후 비밀번호 입력(화면에는 표시되지 않음) 


자주 사용하는 psql 내부 슬래시 명령어 ()

	\l: 데이터베이스 목록 조회

	\c [DB명]: 다른 데이터베이스로 연결 전환
	
		\c [데이터베이스명]: 지정한 데이터베이스로 접속 전환.
			예: \c mydb
		
		\c [데이터베이스명] [사용자명]: 특정 데이터베이스에 지정한 사용자로 접속 전환.
			예: \c mydb myuser
		
		\c - [사용자명]: 데이터베이스는 유지하고 사용자만 변경.
			예: \c - myuser
			
		\c (인자 없이): 현재 접속 정보 표시. 

	\d: 현재 DB 내 테이블 목록 조회
	
	\dt: 테이블 목록 상세 조회
	
	\du: 사용자 및 권한 목록 조회
	
	\q: psql 종료
