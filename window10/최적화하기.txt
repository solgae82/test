		내 활동을 수집하도록 허용 : 체크해제
		이 장치에 내 활동기록 저장 : 체크해제
		활동기록 지우기 버튼 실행
	- 탐색기 실행 > 폴더 옵션 > 일반 > 개인 정보 보호
		빠른 실행에 최근에 실행된 파일 표시 : 체크해제
		빠른 실행에 최근에 실행된 폴더 표시 : 체크해제
	- 바탕화면 우클릭 > 개인설정 > 시작 
		귀찮다. 그냥 버튼들 다 끄자
2. 전원관리
	이걸 설정 안하면, 구형 시스템의 경우 
	유튜브 방송을 볼때 영상과 소리의 싱크가 서서히 맞지 않게 되는 경험을 하게된다.

	제어판 > 전원옵션 > 추가 전원 관리 옵션 숨기기 > 고성능 체크 

	체크된 고성능이 [기본 전원 관리 옵션] 으로 올라와서 선택된 상태에서 우측 '설정 변경' 
	> 고급 전원 관리 옵션 설정 변경(C) 선택
		- 하드 디스크 > 다음 시간 이후에 하드 디스크 끄기 > 설정 : '0' 설정
		- 절전 > 다음 시간 후 절전 모드로 전환 : 사용안함 선택
		- USB 설정 > USB 선택적 절전 모드 설정 > 설정 : 사용안함 선택

3. 내컴퓨터 우클릭 > 속성 > 고급시스템 > 시스템 속성 > 고급
	>성능>시각효과 : 사용자 지정 선택 후 왠만한 것들 모두 체크 해제
	(대부분 애니메이션 기능이므로 성능 저하 발생)

4. OneDrive 끄기 + Widows 디펜더 끄기
	[OnDrive 끄기]: (MS 클라우드 프로그램 같은데...필요 없다..)
	시작>'gpedit.msc'>로컬컴퓨터정책>컴퓨터구성>관리템플릿>Windows  구성 요소
		> OneDrive를 파일 저장소로 사용 방지 클릭
		> 팜업 창에서 '사용' 확인 (안 쓰겠다는 뜻)
	[디펜더 끄기]:
	시작>로컬컴퓨터정책>컴퓨터구성>관리템플릿>Windows구성요소>Microsoft Defender 바이러스 백신
		> Microsoft Defender 바이러스 백신 끄기 클릭
		> 팜업 창에서 '사용' 확인 (안 쓰겠다는 뜻)

5. 제어판 > 키보드 > 문자반복
	재입력 시간(D): 짧게 
	반복 속도 : 빠름

6. 제어판 > 접근성 센터 > 키보드를 사용하기 쉽게 설정 > 입력하기 쉽게 설정 > 필터 키 켜기 
	> 필터키 설정(L) > 필터 옵션 > 반복 키 및 느린 켜기 체크 > 반복 키 및 느린 키 설정(U)
	
	초를 최대한 짧게 해준다

7. 제어판 > 소리 > 통신 탭 : 아무 작업도 안함 체크.

8. 윈도우키 + I > 윈도우 설정 창 
	> 개인 정보
	메뉴들 모두 클릭해보고 왠만하면 다 끄고 비활성하고, 다 지우자

	> 앱
	삭제 할 프로그램들 여기서 삭제하면 된다.(스마트폰 앱 개념)
	
	> 시스템 > 저장공간 > 더 많은 저장소 설정 > 새 콘텐츠가 저장되는 위치 변경
	다른 하드디스크가 있다면 설정한다

	> 업데이트 및 보안 > 배달 최적화 > 다른 PC에서 다운로드 허용 : 끔

	> 개인 설정
		> 테마 > 바탕 화면 아이콘 설정   
			> 바탕화면에 뺄 아이콘 체크, 
			> 아래 쪽 [테마에서 바탕 화면 아이콘 변경 허용(L)] : 체크해제 (요걸 안하면 듀얼 모니터일 경우 바탕화면 아이콘 위치가 맘대로 바뀐다)

9. SSD 일 경우 조각모음 예약 끄기 : SSD에는 조각모음 하면 안된다고 함.
