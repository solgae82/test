디렉티브

  • v-if: true/false 여부로 태그 전체가 삭제되거나, 존재하게 된다
  • v-for: 데이터 갯수만큼 반복 출력
  • v-show: true/false 여부로 태그가 보이거나, 안 보이게 된다 (태그가 사라지는 것은 아니고 false 일때 style="display:none" 적용)
  • v-bind: html 태그 기본 속성과 연결
  • v-on:(이벤트명): 이벤트와 연결한다
  • v-model: 폼 데이터에 설정한다, 폼 데이터 값의 변경 여부에 따라 실시간으로 연결된 메소드 호출이 가능하다.
