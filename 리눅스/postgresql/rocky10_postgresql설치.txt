1. 필수 패키지 설치
	소스 컴파일을 위해 필요한 패키지를 설치합니다.

	dnf groupinstall "Development Tools" -y
	dnf install -y readline-devel zlib-devel openssl-devel wget

2. 소스 코드 다운로드 및 압축 해제 
	PostgreSQL 14 소스 코드를 다운로드합니다. 

	wget https://ftp.postgresql.org/pub/source/v14.12/postgresql-14.12.tar.gz
	tar -zxvf postgresql-14.12.tar.gz
	cd postgresql-14.12

3. 컴파일 및 설치

	./configure --prefix=/usr/local/postgresql-14.12
	make
	make install

4. 환경 설정 및 초기화

	# postgres 사용자 생성
	useradd postgres

	# 데이터 디렉토리 생성 및 권한 설정
	mkdir -p /usr/local/postgresql-14.12/data
	chown -R postgres:postgres /usr/local/postgresql-14.12
	
	# postgres 사용자 변경
	su -u postgres
	
	# 환경변수 설정	
	vi ~/.bash_profile
	..
	export PG_HOME=/usr/local/postgresql-14.12
	export PATH=$PG_HOME/bin:$PATH
	export PGDATA=$PG_HOME/data
	export PGLIB=$PG_HOME/lib
	:wq
	
	# 환경변수 적용
	source .bash_profile
	
5. 데이터베이스 클러스터 생성 

	# 데이터베이스 초기화	 
	initdb --locale='C' \
	--encoding='UTF8' \
	--pgdata='/usr/local/postgresql-14.12/data/svc01' \
	--waldir='/usr/local/postgresql-14.12/wal/svc01' \
	--auth='scram-sha-256' \
	--data-checksums \
	--pwprompt
	
	#현재 사용자가 'postgres' 이고, '--pwprompt' 설정으로 암호를 설정하면 수퍼유저 암호가 설정된 것이다


6. 인스턴스 기동/정지
	#기동
	pg_ctl -D /usr/local/postgresql-14.12/data/svc01 -l logfile start

	#정지
	pg_ctl -D /usr/local/postgresql-14.12/data/svc01 stop

7. 서비스 등록 및 시작
	systemd 서비스 파일을 생성하여 등록합니다.
	
	vi /etc/systemd/system/postgresql-14.12.service
	파일 내용:
	
		[Unit]
		Description=PostgreSQL 14 database server
		After=network.target

		[Service]
		Type=forking
		User=postgres
		Group=postgres
		Environment=PGDATA=/usr/local/postgresql-14.12/data
		ExecStart=/usr/local/postgresql-14.12/bin/pg_ctl start -D ${PGDATA}
		ExecStop=/usr/local/postgresql-14.12/bin/pg_ctl stop -D ${PGDATA}
		Restart=always

		[Install]
		WantedBy=multi-user.target
		bash

	# 서비스 시작
	systemctl daemon-reload
	systemctl enable postgresql-14.12
	systemctl start postgresql-14.12
