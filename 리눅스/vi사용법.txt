vi  ---------------  사용법



1. 명령모드에서 하기

--라인이동
j : 다음행
k : 이전행
h : 이전 한글자 이동
l : 이후 한글자 이동


$ : 행 끝으로 이동
^ : 행 처음으로 이동(공백이 아닌 행에 첫문자 위치로)
0 : 행 처음으로 이동(무조건 제일 처음 행 위치로)

--문서이동
H : 현재 화면에서  제일 상단 이동
M : 현재 화면에서  중간 위치로 이동
L : 현재 화면에서 맨 마지막으로 이동
20G : 20줄 위치로 이동

ctl+f : 다음 화면 이동
ctl+b : 이전 화면 이동


--취소
u : 이전 작업 취소

--복사/잘라내기/붙여넣기
yy : 행 복사, 3yy는 3줄 복사
dd : 삭제겸 잘라내기 3dd 는 3줄 잘라내기
p  : 커서가 위치한 곳에 붙여넣기
x  : 커서 위치의  한 글자 잘라내기




2. 마지막 행 모드에서 하기

-- 검색하기
/문자열 : 현재 커서가 위치한 곳에서 아래로 이동하면서 [문자열]검색, /css <-css 문자 검색
?문자열 : 현재 커서가 위치한 곳에서 위로 이동하면서 [문자열]검색, ?css <-css문자 검색
n	: 다음 문자열 검색 \
N	: 진행 방향엣 역순으로 검색

-- 행번호 표시하기
:set nu <- 행번호 표시
:set nonu <- 행번호 표시 감추기

-- 복수파일 편집
:r 파일명 <-현재 커서 위치에 파일내용을 복사해서 붙여넣음..
:e 파일명 <- 다른 파일을 vi로 연다(기존 파일을 w로 저장하고 해야함.)
:n 다음파일 <- 복수 파일로 열었을 경우 다음 파일로 이동, 한번밖에 이동 못함.
		예를 들어 vi a.txt b.txt c.txt파일 열고 a->b->c 이동하면 끝.
		(편집했다면 w로 저장하고 :n해야함 or :wn [:e 와 같음])

-- 작업 도중 shell 명령 이용하기
:! pwd <- 쉘 명령 실행하기, 실행 후 enter하면 다시 vi모드로 돌아옴(1회성)
:sh <-아예 쉘모드로 나가서 여러 작업 후 , exit를 하면 다시 vi모드로 돌아옴(긴 작업시)
