1.instance client 또는 oracle client 설치
	oracle client 설치디렉토리는 자동으로 path에 등록된다.

2.ORACLE_HOME, TNS_ADMIN 환경변수 생성 (toad가 사용하는 환경변수)

	ORACLE_HOME = C:\oracle\11.2.0\client_1
	TNS_ADMIN = %ORACLE_HOME%\network\admin

3. Path 환경변수에 ORACLE_HOME 환경변수 등록(toad와 상관없지만 그냥 등록)
	(sqlplus cmd 실행시 바로 쓰기좋게 , oracle client는 자동등록)
	
	path = %ORACLE_HOME%;

4. TNS_ADMIN 경로에 tnsnames.ora 파일생성

	ora10 =
	  (DESCRIPTION =
	    (ADDRESS_LIST =
	      (ADDRESS = (PROTOCOL = TCP)(HOST = 192.168.0.3)(PORT = 1521))
	    )
	    (CONNECT_DATA =
	      (SERVER = DEDICATED)
	      (SERVICE_NAME = ora10)
	    )
	  )
5. toad 콤보박스에 출력되는지 확인.
