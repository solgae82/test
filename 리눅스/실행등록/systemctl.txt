CentOS 7이상부터 service와 chkconfig가 <systemctl>로 통합되었음.

--서비스실행
	systemctl start 서비스명
	systemctl restart 서비스명
	systemctl stop   서비스명

--자동실행 등록

	systemctl enable 서비스명 
	systemctl disable 서비스명 

	추가로 실제 서비스가 자동실행으로 적용됐는지 확인은
		systemctl is-enable 서비스명 

	자동실행 서비스 목록을 보고 싶을때 
		systemctl list-unit-files --type=service

httpd 예제
----------------------
1.서비스 시작
	systemctl start httpd.service

2.서비스 중지
	systemctl stop httpd.service

3.서비스 재시작
	systemctl restart httpd.service

4.서비스가 구동중이었을 경우에만 재시작
	systemctl try-restart httpd.service

5.설정 재구동
	systemctl reload httpd.service

6.서비스 구동 여부 확인
	systemctl status httpd.service
	systemctl is-active httpd.service

7.모든 서비스의 상태 표시
	systemctl list-units --type service --all

----------------------
1.서비스 활성화(부팅시 자동 구동)
	systemctl enable httpd.service

2.서비스 비활성화
	systemctl disable httpd.service

3.서비스의 활성화 여부 표시
	systemctl status httpd.service
	systemctl is-enabled httpd.service

4.모든 서비스의 현재 활성화 여부 표시
	systemctl list-unit-files --type service

5.지정한 target 이후에 시작하는 서비스를 표시
chkconfig --list => systemctl list-dependencies --after

6.지정한 target 이전에 시작하는 서비스를 표시
chkconfig --list => systemctl list-dependencies --before
