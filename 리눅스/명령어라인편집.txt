- 명령줄 한칸 이동
	전진 (오른족 한칸 이동): ctrl + f 
	후진 (왼쪽 한칸 이동): ctrl + b  
	
- 명령줄 처음, 끝 이동
	
	처음 이동 :  ctrl + a
	끝 이동: ctrl + e
	
- 단어 단위 이동
	meta 키란? : ESC , Alt 

	전진: alt + f
	후진: alt + b
	-----------
	전진: ESC 누르고 뗀 뒤 f
	후진: ESC 누르고 뗀 뒤 b
	
- 문자 삭제
	후진 삭제: Backspace
	후진 삭제: ctrl + h
	
	전진 삭제: Delete
	전진 삭제: ctrl + d
	
	후진 단어삭제: ctrl + w

	붙여넣기: ctrl + y
