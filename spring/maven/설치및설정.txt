메이븐 설치
	1. 최신 다운로드 페이지에서 .zip 파일을 받는다
	https://maven.apache.org/download.cgi

	2. 적절한 위치에 압축 풀기
		D:\maven 에 압축 풀었다고 가정하고..
			
			.bin  #바이너리 프로그램 파일 디렉토리
			.boot 
			.conf # 설정 디렉토리
			.lib # 라이브러리 디렉토리
			... 기타등등 파일
		
		
	3. 윈도우 환경변수 설정
		MAVEN_HOME 변수 만들기
			내컴퓨터>속성>고급시스템설정>고급>환경변수>시스템변수>새로만들기
			변수이름 : MAVEN_HOME
			변수값 : D:\maven 
		PATH 변수에 등록하기
			내컴퓨터>속성>고급시스템설정>고급>환경변수>시스템변수>Path>새로만들기
			변수값 : %MAVEN_HOME%\bin

	4. cmd 창에서 환경설정 확인
		mvn -v #메이븐 버전 확인
