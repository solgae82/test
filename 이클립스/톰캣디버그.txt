
일반 자바 프로젝트는 java파일에 break point 찍고 ctrl + F11 실행으로 디버그 하면 된다

웹프로젝트할때는 톰캣 실행에 디버그모드(ctrl + alt + D) 실행하는 방법으로 할 수 있다.

	java 파일에 break point 걸고

	서버탭 > 톰캣서버 우측 마우스 클릭 > debug 실행

	브라우저에서 해당 페이지 실행

진행 단축 키
	
	※ F6 키가 중심이다

	F5 : 객체나 메서드 등이 있으면 더 깊이 들어가기(depth)..계속
	F6 : break point 걸리면 한 라인씩 진행
	F7 : 들어간 (depth) 있다면 빠져나온다 
	F8 : 다음 break point로 이동
