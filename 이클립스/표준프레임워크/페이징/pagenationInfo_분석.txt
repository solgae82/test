페이지 클래스에서 쓰이는 변수값들 해석

모든 변수들은 int 타입이다.

--------- 필수 입력 설정값
	currentPageNo;		// 현재 페이지 번호
	recordCountPerPage;	// 한페이지에 게시될 게시물 수
	pageSize;		// 페이지 리스트(한번에 표시될 페이지 갯수) 갯수, 페이지 블록
	totalRecordCount;	// 전체 게시물 총갯수


---------계산 값
	아래 예제 값들은 
		recordCountPerPage = 10;
		pageSize = 10; 

	으로 가정한다

	총 페이지 갯수 값

		totalPageCount : 총 페이지 갯수 정수 값.. 

	변수 없이 출력하는 계산 값
		public int getFirstPageNo() {
			return 1;
		}

		public int getLastPageNo() {
			return 총 페이지갯수;
		}


	현재 페이지 리스트(블록)에서 쓰는 변수

		firstPageNoOnPageList : 1 , 11, 21, 31 ..
		lastPageNoOnPageList : 10, 20, 30, 40 ..


	SQL에서 쓰는 rownum 값 변수

		firstRecordIndex : 0 , 10, 20, 30 ..
		lastRecordIndex : 10, 20, 30 ...
