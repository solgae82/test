주요 WSL 명령어 및 동작:

wsl: 기본 리눅스 배포판을 실행.
wsl --install: WSL 및 기본 리눅스 배포판(Ubuntu) 설치.
wsl -l -v (또는 --list --verbose): 설치된 배포판 목록과 WSL 버전(1 또는 2) 확인.
wsl -d <배포판이름>: 특정 배포판으로 WSL 실행.
wsl --shutdown: 실행 중인 모든 WSL 배포판 종료.
wsl --update: WSL 커널 업데이트.
wsl --unregister <배포판이름>: 설치된 배포판 삭제 및 데이터 초기화.
wsl --help : 모든 사용 가능한 명령어 출력
wsl --set-default-version 2 : 디폴트 wsl 버전 2 설정
wsl --set-version Ubuntu-24.04 2 : 설치된 Ubuntu-24.04 wsl 버전 2로 변경
wsl --shutdown : 실행중인 모드 배포판 끄기

# 배포판 로그인
wsl --user root # 기본 배포판 root로 로그인
wsl -d Ubuntu24-04 --user root # Ubuntu24-04 지정해서 로그인

# 배포판 삭제
wsl --unregister Ubuntu-24.04
