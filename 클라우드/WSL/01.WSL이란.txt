WSL이란
	WSL(Windows Subsystem for Linux)은 윈도우 10/11 및 서버 환경에서 
	별도의 가상 머신(VM)이나 듀얼 부팅 없이 리눅스(Ubuntu, Debian 등)를 직접 실행하는 기능
	
WSL 핵심 특징
	가벼운 가상화: VM보다 적은 리소스를 사용하여 성능 저하가 거의 없습니다.
	간편한 설치: Microsoft Store에서 Ubuntu, Kali Linux 등 원하는 배포판을 설치할 수 있습니다.
	통합 환경: 윈도우 파일 시스템(C:\)에 리눅스에서 바로 접근할 수 있으며, VS Code와 같은 윈도우 앱과 연동성이 뛰어납니다.
	WSL 2: 실제 Linux 커널을 사용하여 도커(Docker) 등 더 높은 호환성과 속도를 제공합니다. 

------------------- 윈도우즈10, 11에서  wsl 설치

1.윈도우 기능 켜기/끄기 실행
	방법1
		실행창(Win+R) 입력: optionalfeatures 을 입력하거나,
		시작메뉴>'윈도우 기능 켜기/끄기' 입력 후 실행> (창 열림)
			(v) Linux용 Windows 하위 시스템
			(v) 가상머신 플랫폼
			
		두개 체크 후 확인
		
	방법2
		(powershell 관리자 권한으로 열고 명령어로 실행)
		
		# Linux용 Windows 하위 시스템 (활성화)
		dism.exe /online /enable-feature /featurename:Microsoft-Windows-Subsystem-Linux /all /norestart
		
		# 가상머신 플랫폼 (활성화)
		dism.exe /online /enable-feature /featurename:VirtualMachinePlatform /all /norestart

	설정 후 컴퓨터 재시작
	
2. Microsoft Store에서 Windows Subsystem for Linux를 설치한다
	https://www.microsoft.com/store/productId/9P9TQF7MRM4R?ocid=pdpshare
	
3. 설치 후 도스창에 명령대로 실행
	
	wsl --list --online # 설치가능한 리눅스 배포판 온라인 리스트 보기
	
		다음은 설치할 수 있는 유효한 배포 목록입니다.
		'wsl.exe --install <Distro>'을 사용하여 설치합니다.

		NAME                            FRIENDLY NAME
		Ubuntu                          Ubuntu
		Ubuntu-24.04                    Ubuntu 24.04 LTS
		..
		
	
	wsl --install Ubuntu-24.04 # 최신 배포판으로 설치
		
		#설치 후 생성할 기본 사용자와 패스워드 물어봄. 입력 생성 후 계속 설치된 Ubuntu-24.04 실행 상태
			
		# 우분투 최신 상태 만들기
		sudo apt update
		sudo apt upgrade 
		
		exit
		
	#다시 Ubuntu-24.04 들어가고 싶다면
	wsl -d Ubuntu-24.04
	
	# 실행중인 모든 wsl 배포판 종료
	wsl --shutdown 
		
4. 설치된 배포판과 wsl 버전 확인

	wsl -l -v 
	
	 NAME			STATE           VERSION
	 Ubuntu-24.04		Stopted		2
	
※ wsl 버전 1에서 2로 업그레이드 
	
	#아래 처럼 버전이 1이라면 2로 업그레이드 해야한다
	wsl -l -v 
	
	 NAME			STATE           VERSION
	 Ubuntu-24.04		Stopted		1


	wsl --install # 처음 설치할때 요걸로 , 윈도우 11에서 2버전으로 설치됨
	# wsl --update # wsl 치된 버전이 1이라면 최신 버전으로 업그레이드
	
	# 이미 설치된 우분투 wsl 버전 2로 변경
	wsl --set-version Ubuntu-24.04 2
	
	# 기본 wsl 버전 2로 설정
	wsl --set-default-version 2
	
------------------- wsl 명령어 모음

주요 WSL 명령어 및 동작:
wsl: 기본 리눅스 배포판을 실행.
wsl --install: WSL 및 기본 리눅스 배포판(Ubuntu) 설치.
wsl -l -v (또는 --list --verbose): 설치된 배포판 목록과 WSL 버전(1 또는 2) 확인.
wsl -d <배포판이름>: 특정 배포판으로 WSL 실행.
wsl --shutdown: 실행 중인 모든 WSL 배포판 종료.
wsl --update: WSL 커널 업데이트.
wsl --unregister <배포판이름>: 설치된 배포판 삭제 및 데이터 초기화.
wsl --help : 모든 사용 가능한 명령어 출력
wsl --set-default-version 2 : 디폴트 wsl 버전 2 설정
wsl --set-version Ubuntu-24.04 2 : 설치된 Ubuntu-24.04 wsl 버전 2로 변경
wsl --shutdown : 실행중인 모드 배포판 끄기
