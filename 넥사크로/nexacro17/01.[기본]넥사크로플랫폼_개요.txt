01. [기본]넥사크로플랫폼 개요

	기술지원 URL
	http://support.tobesoft.co.kr
	

	교육자료: Education > 교육자료 25번

	개발툴(Studio) : Download > nexacro platform17> 게시판 컬럼 Products> 'Developer' 버전 다운로드 
		(nexacro17_SetupStudio.exe)
	
		개발용 라이선스 사용기간 : 60일 (매월 1일 기준,오래 쓰려면 1일에 받아야 함)

	교육목적
		넥사크로 플랫폼 이해
		제품구조 및 실행환경
		컴포넌트 기능과 바인딩
		사원정보 프로그램 완성(서비스와 연동)
	
	넥스크로 개요
		자바스크립트 기반의 웹 UI 솔루션
		HTML5 기반
		다양한 디바이스에 적용 가능(pc,태블릿,모바일 등)
		웹브라우저 뿐만 아니라 nexacro 자체 브라우저(native환경이라고 함)로도 play 가능.
		(웹환경, 네이티브 환경을 모두 만족시킨다는 뜻)
		
	넥사크로 스튜디오
		하나의 개발툴에 개발자, 디자이너,퍼블리셔 등이 사용가능한 4GL 개발도구
		40여개 비지니스 컴포넌트
		풍부한 Grid기능 (Grid Contents Editor)
		다양한 UI표현의 Dashboard등
		스크린 애뮬레이터 제공
		개발편의성 강화(code snipet)
			
	넥사크로 Deploy 순서
		Studio 개발작업 > form.xfdl > (Generate) > form.js
		
	Unified framework(JS) 
		: Nexacro library
		
	동작방식
		(일반브라우저) : 
		form.js + Unified framework(JS) + 웹브라우저별 렌더링 환경
		
		(NRE : Nexacro Runtime Environment)
		form.js + Unified framework(JS) + nexacro 자체 브라우저 
		
	nexacro 브라우저
		: 브라우저가 제공하지 못하는 여러기능들을 추가적으로 지원한다
		엔진: Script engine (V8) 
		
		추가지원 기능들
		Device API, Local DB, Plug-in, Local File
		
	
	project 생성
		1. start페이지 > NEW
		2. 메뉴 > File > New > Project
		3. 바로가기 아이콘 > New Project 
		4. 단축키 : ctrl+shift+N
		
	
		Project Wizard 창 > New Project
			
			Project 
				Application
			Template  # 17.1에 있는 템플릿 메뉴 
				MDI (Division)
				MDI (Frameset)
		
		
		Destop , Tablet, Phone 선택창
		
		
		Project Name : nxp 
		Location : C:\
		
		Frameset Template : 메뉴 구성을 어떻게 할 것인가 선택
		Detail's Setting (Optional) :선택된 프레임을 세밀하게 조정하는 메뉴
		
		
		프로젝트 생성 후 Project Explorer 창에 기본 프로젝트 구조가 설정됨. 
		
	Project Exploer 메뉴
		Project 'nxp'
		  Environment
			ScreenDefinition
				Desktop_...
				(dektop 이외에 모바일도 선택했다면 두개의 메뉴가 나올 것이다)
				
			Variables : 싱글사인온 같은 기능(시스템과 시스템 공유)을 위한 변수를 지정해할때 쓴다
				브라우저 localStorage 저장된다.
				브라우저를 삭제하지 않는한 계속 남아 있기 때문에, 변수를 삭제하는 코드도 같이 코딩해야한다.
			Cookies : 쿠키변수 정의, 서버 세션 키등을 정의,WAS 세션변수(쿠키)와 같이 한다.
			HTTP Header : http header 정보 추가할때 정의, 방화벽등의 정책에 필요한 변수등을 정의할때 사용.
			Script : Variables, Cookies, HTTP Header 등을 javascript 로 세밀 설정할때 정의.
		  
		  TypeDefinition
			Objects: 40여개 컴포넌트 리스트, 등록,삭제(숨기기)등  
				Modules : .json 파일들, 라이브러리들 모음, 프로젝트에 추가, 삭제
				Objects : 현재 프로젝트에 사용가능 컴포넌트들을 표시
			Services:
				Resource Service : 디자이너 영역
				User Service : 개발자 영역
			ProtocolAdapters: 별도의 프로토콜 사용하고 매핑할 수 있는 영역
			
			DeviceAdaptors: 마우스,키보드 외에 음성인식,제스처등 외부 인터페이스 추가 가능
			
		  Application Infomation
			Application Variables (글로벌 영역
				Datasets : 글로벌 데이터 셋 정의 (2차원 형태의 데이터를 담을 수 있다)
				Variables : 어플리케이션 종료 후 사라지는 글로벌 변수 정의
			Applications
				Application_Desktop : 템플릿 선택시 나오는 화면 결과물 frame들이 모여 있는 곳
