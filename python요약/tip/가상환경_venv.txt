
운영체제에 설치된 여러 버전 파이썬 중 3.12 버전을 설정해서 가상환경 셋팅하는 방법이다

vscode 설정 기준이다.
(vscode에서 일단 파이썬 파일 한개는 열고 가상환경 설정 시작해야 우측 아래에 선택 화면이 뜬다.)

1. vscode 시작

2. 프로젝트 폴더 선택

3. 아무 파이썬 파일하나를 만들어서 열던지, 있던 거 열던지 한 후 다음단계 진행
	test.py # 만들어 연다
	
3. vscode 터미널 열어서(powershell)

	python -3.12 -m venv venv
	
	#프로젝트 폴더에 venv 폴더가 생성되었는지 확인
	
4. 가상 환경 활성화 (매번 새로 열때마다 해야함)
	
	venv\Scripts\activate
	
	#가상 환경이 활성화되면 터미널 창 앞에 (venv)라는 기호가 추가됨
	
5. 만약 (venv) 가상환경 프롬프트가 나오지 않는다면, powershell 실행 정책 변경
	
	- powershell 관리자 모드로 실행
	- 'Set-ExecutionPolicy RemoteSigned -Scope CurrentUser' 실행
	
	설정 변경 후 '4번' 다시 실행
	
6. 가상 환경이 설정되면 vscode 우측 아래에 창이 뜨면서 가상환경 선택할거냐고 물어볼 것이다.
	
	'We noticed a new environment has been created.
	Do you want to select it for the workspace folder?'
	
	(Yes 버튼 클릭)

	# 위와 같은 메시지가 뜨지 않는다면 수동으로 버전 선택해야 한다.
	
7. 파이썬 가상환경 수동 버전 선택 하기
	
	- vscode 우측 아래 창에 'Python 3.12(64-bit)' 등의 메시지를 선택하거나,
	또는 단축키 ctrl + shift + p 를 누른다
	
	- 선택 박스 중 'Python: Select Interpreter' 를 선택해서 
	원하는 파이썬 버전 선택해서 사용
	
	ex) Python 3.12 (64-bit) C:\python\3.12.4\pythoe.exe
	

8. 가상환경 비활성화
	
	$ deactive
	
	# (venv) 프롬프트가 사라지고 파이썬 기본 글로벌 환경으로 바뀐다
	
	
※ vscode 종료하고 다시 시작하면 다시 가상환경 실행하고,파이썬 버전 선택 후 작업해야한다

	venv\Scripts\activate #(가상환경 활성화)
	
	ctrl + shift + p
	(Python 3.12 (64-bit) C:\python\3.12.4\pythoe.exe) #선택
